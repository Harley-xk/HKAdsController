# DomobAd-iOS-SDK
多盟广告iOS SDK

# 简介
此为非官方整理，代码文件来自多盟官方网站，本人仅对部分头文件错误进行修改，其余资源文件及静态库未进行任何修改。

# 版本
4.4.8 [更新时间:2014-12-12]

# 兼容平台
iOS 5.0及以上

# 使用
可手动下载后参照`UserGuide.pdf`进行配置

或

使用[cocoapods](http://cocoapods.org/)进行安装,在`podfile`中添加如下代码

> `pod 'DomobAdSDK'`

或

> `pod 'SS-DomobAdSDK'`
